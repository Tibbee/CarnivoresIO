CarnivoresIO

Blender extension for importing, editing, and exporting .3df, .car, and .3dn models from the classic Carnivores game engine. Full animation, rigging, and audio support.

Download Latest Install via Dev Channel Full Documentation Source Code

Supported Formats

.3dfStatic models, import & export
.carAnimated models, import & export
.3dnMobile/HD static export
.vtlAnimation export

Quick Start

1

Install

Download the latest release .zip and drag it into Blender's Extensions preferences, or add the dev channel URL for auto-updates.

2

Import

Use File > Import > Carnivores Engine to load .3df or .car files. Enable Flip Handedness for the game's coordinate system.

3

Rig & Export

Click Reconstruct Rig from Owners in the sidebar to build an armature, link sounds, tweak KPS, then export.

Features

Skeleton Reconstruction

Symmetry-aware MST algorithm infers bone hierarchy from vertex groups automatically.

Animation Pipeline

Vertex animations become Shape Key Actions with NLA strips. KPS timing for exact engine reproduction.

NLA Sound System

Link .wav files to actions for synchronized audio during timeline scrubbing.

Face Flag Tools

Viewport visualization, bitmask selection, and bulk modification of engine face flags.

Validation & Repair

Import checks for limits, UV ranges, bone cycles, and texture integrity with auto-repair.

Weight Smoothing

Laplacian smoothing with joints-only mode for organic deformation.

Compatibility

Blender 4.2 – 6.0 Windows / macOS / Linux NumPy (bundled) GNU GPL v3

Requires Blender 4.2+. Blender 3.x is not supported due to the Extensions API.
